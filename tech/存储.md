
### 什么是存储

存储元件都雷打不动地分为两大派系：RAM（内存派）和ROM/Flash（外存/闪存派）。
  -  **⚡ RAM 易失性存储 (内存派)**
        * **特点**：速度极快，断电数据立刻消失。
        * **SRAM (静态内存，CPU缓存)**
        * **DRAM (动态内存，大容量)**
            * **DDR (电脑/服务器)**：DDR4 / DDR5
            * **LPDDR (手机/轻薄本)**：LPDDR5X / LPDDR6
            * **GDDR (显卡图像)**：GDDR6 / GDDR7
            * **HBM (先进封装，芯片连体楼)**：HBM3e / HBM4
  -   **🔒 ROM/Flash 非易失性存储 (外存)**
        * **特点**：速度较慢，断电数据长效保存。
        * **ROM 原始只读**：已基本淘汰
        * **NOR Flash (利基闪存，存开机死代码)**
        * **NAND Flash (大容量闪存，主力仓库)**
            * **2D NAND (平面闪存)**：已淘汰
            * **3D NAND (垂直叠楼闪存)**
                * **eMMC/UFS (手机闪存)**
                * **SSD固态硬盘 (电脑/服务器)**


DRAM（动态随机存取内存）—— 真正的“内存基本盘”，由一个晶体管 + 一个微型陶瓷电容器组成。因为电容会漏电，它每秒需要刷新充电几百次才能维持数据 

HBM（高带宽内存）：DRAM 家族的究极进化体。 抛弃平铺，利用先进封装把十几层 DRAM 芯片像盖楼一样垂直堆叠 [longfei_fiber_market]，用万部 TSV 铜电梯连通，直接焊在计算核心旁边，打破内存墙。


#### DRAM生产流程

与半导体流程相似，存储作为下游应用，在模组、主控芯片等方面和其他应用有差异

1. 晶圆制备- 大地基 
   - 高纯多晶硅：必须过滤掉千万分之一的杂质，提炼出纯度到顶的特种原材料
   - 单晶硅锭（拉晶）： 将原材料熔化，像拉麦芽糖一样拉出一根晶格完美一致、几百公斤重的圆柱形水泥大柱子。
   - 切片与抛光： 用比头发丝还细的锯子，把这根大柱子横着切成一片片薄片，并把表面打磨得像镜面一样光滑
2. 前道工艺（Fab）： 我们要在这个地基上，同时盖出几十亿个包含“一扇防盗门（晶体管） + 一个蓄水池（电容）”的标准高科技房间。
   - 光刻（DUV/EUV）： 拿激光“投影仪”往地上画图纸
   - 刻蚀与薄膜沉积（ALD）：按照强光照出来的图纸痕迹，用强酸和等离子体在硅片上垂直轰击、向下挖出几百亿个纳米级的“直立储水井（电容沟槽）。随后用化学气体像刷漆一样，在井壁上均匀抹上一层只有几个原子厚度的高纯陶瓷水泥（High-K材料）防止漏水，再填入金属做导线
   - 重复 500+ 道工序 ： 清洗、挖坑、砌墙、铺电线（金属字线/位线），这个过程要反复重来 500 多次，大楼越盖越高。
   - 晶圆测试（Wafer Sort）：质量大检查，贴“合格/违建”标签

 3. 后道工艺-打包发货、组装
    - 切割 把大晶圆上连在一起的几十亿间房子，切成一块块独立指甲盖大小的、功能完备的（Die/芯片颗粒内核）。
    - 封装：封测厂把它粘在底座上，用微型锡球把它的引脚连出来，最后灌入黑色的硬胶（环氧树脂）整体注塑死死包住。
    - 模组组装：把这些黑色的芯片颗粒、小电容电阻、电源芯片，整整齐齐地焊接在一块长条形的绿色/黑色电路板（PCB）


#### HBM先进封装

HBM最引人瞩目、最暴力的地方确实在后道先进封装上

- 普通的 DDR 内存晶圆在晶圆厂里做完 1T1C（晶体管+电容） 就可以直接出厂了。但 HBM 晶圆在前道流水线上，必须多承受几道变态的高难度物理折磨：
  1. 打通垂直电梯井（TSV刻蚀 ── 发生在晶圆厂前道/中段）：在长鑫存储或三星的晶圆厂里，内存电路刚刻好，就必须动用变态的高能等离子体刻蚀机，在完整的硅晶圆上，垂直向下轰击钻出数万个直径仅有几微米、深达几十微米的超级深孔（TSV） [longfei_fiber_market]。这在普通 DDR 内存的制造中是完全没有的工序。
  2. 铺设铜电梯（TSV金属化 ── 发生在晶圆厂前道/中段）：晶圆厂必须通入化学气体，在这些几万个深孔内部，用电镀手艺填满超高纯度的金属铜，做成垂直导电通道 [longfei_fiber_market]。
  3. 重新布线层（RDL材料加工）：晶圆厂必须在晶圆的表面和背面，用光刻技术额外长出一层高密度的微型铜引脚网络（RDL，再布线层），用来给上下层大楼精准对齐分配信号。

- 后道工艺的走向直接发生了解构性的分化，这也是普通代工（SMT）向先进封装（3D 混合键合）

    | 制造与组装步骤 | 普通内存条（DDR4 / DDR5） | HBM 内存大楼（HBM3e / HBM4） |
    | :--- | :--- | :--- |
    | **1. 硅片削薄 (Thinning)** | 稍微磨一磨，保持芯片厚度，防止芯片太脆折断 | **极限削薄！** 用纳米磨盘把屁股削到 **30-50微米（比头发丝还薄）**，直到把前道埋的铜电梯屁股完全露出来。 |
    | **2. 芯片切片 (Dicing)** | 用传统钻石刀切成一颗颗独立的指甲盖大小裸晶片（Die）。 | 同样切片，但因为太薄，必须用高精度激光或特种保护膜，防止切片时芯片粉碎。 |
    | **3. 大楼合体 (Stacking)** | **平铺摆地摊！** 拿塑料外壳把小晶片包成黑方块（FBGA封装），**用焊锡膏平铺焊接在树脂电路板（PCB）上**。 | **3D垂直叠罗汉！** 12层芯片面对面精准对齐，采用**混合键合（Hybrid Bonding）**材料技术，**让上下两层芯片的铜原子和二氧化硅分子在室温下直接扩散融合、凝结成一整块硬石**。不加任何焊锡，把楼高压到极限。 |
    | **4. 特种材料护航 (Materials)**| 芯片外围焊上普通的 MLCC 陶瓷电容稳压即可。 | 封装内部要盖上宇宙导热第一的 **[CVD金刚石散热薄膜]** 摊散热斑，并并排焊接在 **[高纯度玻璃基板（沃格光电）]** 地基上，缝隙塞满皮秒响应的 **[硅电容]**。 |
    | **5. 最终成品 (Final Product)** | 变成一根一根的**长条形内存条**，消费者可以随时拆卸更换。 | 变成一个黑色的半导体大方块，**直接被先进封装（CoWoS）和 GPU 大脑焊死在同一个大底座上**，共享呼吸。 |
